Installation

  • Go to http://www.cygwin.com/

  • Chose Intall or Update now icon.

  • Setup.exe is copied to, for example c:\downloads\cygwin

  • Then execute setup.exe

  • Pretty much straightforward, until the packages.

  • Choose carefully. The doc helps.

  • It installs as advertized.

Configuration

  • Open a Cygwin session.

  • Ensure DISPLAY is set to 127.0.0.1:0.0 (or whatever is appropriate).

  • xhost the X Client, for example xhost +mine

Don't forget to set DISPLAY on XClient to the IP of the X Server.

To start the X Server

I made a desktop short cut to /usr/X11R6/bin/startxwin.bat. This seems to be OK.

To stop the X Server

When you are done using Cygwin/XFree86, press Alt - F4 to shutdown the X Server; this is the default key combination to shutdown the X Server. The -[no]winkill and -[no]unixkill parameters can change the set of key combinations that will shutdown the X Server. Using the -unixkill allows the Ctrl - Alt - Backspace key combination to shutdown the X Server, in addition to the default Alt - F4 key combination
