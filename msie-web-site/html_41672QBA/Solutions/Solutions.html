Java Application Launcher for Windows

 

This program allows a Windows user to run a Java Application from a shortcut, or automatically from Explorer. It requires a configuration file. I explain the usage through an example.

 

Download the program and copy to c:\windows. Actually, it just needs to be in your PATH, but c:\windows will get it done.

 

Create file app.jlv and enter the following: 

env,CLASSPATH,deploy
// java App
run,javaw App

where:

  • env sets an enviroment variable. Will only be set in the environment of the Java Application.

  • run sets the command line.

  • all other command types are ignored.

Note:

  • the above example assumes the class files are in the deploy subdirectory of the directory that stores the jlv file.

  • you may enter the usual command line parameters for the JVM.
  • you may enter any environment variable, including JAVA_HOME and PATH if you wish.
  • use javaw for no console, java if you require a console.

Start Explorer. Locate the file app.jlv. Select the file and right click. Select Open.

Windows will complain that it cannot open the file. You have the file that knows how to handle this type of file. You just need to tell Windows which program to run.

Select the "Select the program from a list" radio button and select OK.

A list of programs will appear. The program you need, namely jvjava.exe will not be in the list. Tick the "Always use the selected....." checkbox. Click on the Browse button.

Navigate to c:\Windows and select jvjava.exe.

 

From now on, whenever you open a jlv type file from explorer, the Java Application Launcher will automatically run the Java Application.

 

To create a short cut, locate the app.jlv file, right click, create short cut, drag the short cut to your desktop, and rename the short cut as per your requirements.
