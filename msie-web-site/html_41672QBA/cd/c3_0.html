Card Sort II

My original card sort had too many top level categories. This is an attempt to reduce that number to a manageable level for the main page. For the current assignment, I plan to create two card sorts based on my previous work.

  • The first is close to the original but with fewer links from the home page.

  • The second has only 11 links from the home page.

I plan to use Design 2 for the moment. With fewer links:

  • It will be relatively easy to add a few links if I have to.

  • The page will probably be easier to design.
