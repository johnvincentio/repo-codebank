General

  • host – look up host names using DNS.

  • traceroute – print the route packets take to a network host. Includes host, IP and 3 timings. Notes packets dropped.
  • nmap – network exploration tool and security scanner. Find out which hosts are up and what services they are offering.
  • wget – network utility to get files from the WWW, using HTTP and FTP.
  • telnet – user interface to the telnet protocol. Used to communicate with another host
  • nc – basic web server.
